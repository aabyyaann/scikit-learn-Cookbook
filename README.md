# Repository Overview
Repository ini berisi hasil pengerjaan **Tugas 2 Enrichment for Machine Learning Classes**, yaitu **reproduksi kode dan pendalaman teori** dari buku:

> **scikit-learn Cookbook (O’Reilly)**

Setiap chapter direproduksi ke dalam **notebook `.ipynb` terpisah**, disertai:
- ringkasan materi
- penjelasan teori dalam bahasa Indonesia
- implementasi kode menggunakan **Python** dan **scikit-learn**
- visualisasi, evaluasi, dan interpretasi hasil eksperimen

Repository ini dibuat sebagai bentuk dokumentasi pembelajaran terstruktur mengenai workflow machine learning menggunakan **scikit-learn**, mulai dari preprocessing data, supervised learning, unsupervised learning, evaluasi model, hingga deployment sederhana.

---

# Main Objective
Tujuan utama repository ini adalah untuk memperdalam pemahaman konseptual dan kemampuan implementasi praktis terkait **Machine Learning menggunakan scikit-learn** melalui pendekatan:

1. **Code Reproduction**  
   Mereproduksi contoh-contoh kode dari buku ke dalam notebook yang dapat dijalankan ulang.

2. **Theoretical Deep-Dive**  
   Menjelaskan teori penting pada setiap chapter dengan bahasa yang lebih sederhana dan terstruktur.

3. **Hands-on Experimentation**  
   Menjalankan eksperimen, memodifikasi parameter, membandingkan model, dan mengamati hasil evaluasi.

4. **Documentation and Reproducibility**  
   Menyusun notebook dan README agar mudah dipahami, dipelajari ulang, serta direproduksi melalui GitHub.
